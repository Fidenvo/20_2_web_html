HTML Head Checklist

Hướng dẫn cấu hình phần head chuẩn cho mọi trang HTML

1. Meta Charset

Khai báo bộ mã ký tự UTF-8 để hiển thị tiếng Việt đúng dấu.

<meta charset="UTF-8">

2. Meta Viewport

Giúp trang hiển thị đúng trên điện thoại và máy tính bảng.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

3. Title

Tiêu đề hiển thị trên tab trình duyệt.

<title>Tên trang</title>

4. Liên kết CSS

Kết nối file CSS bên ngoài để tạo kiểu cho trang.

<link rel="stylesheet" href="style.css">

5. Liên kết JS

Đặt thẻ script cuối body để nội dung tải trước, JS chạy sau.

<script src="js/script.js"></script>

6. Bình luận

Dùng bình luận để đánh dấu các khu vực trong code cho dễ đọc.

<!-- Đây là bình luận -->

7. Thụt dòng

Thẻ con phải thụt vào một mức so với thẻ cha, thể hiện quan hệ rõ ràng.
